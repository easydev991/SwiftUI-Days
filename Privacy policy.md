Политика конфиденциальности для приложения **Счётчик дней**

# 1. Введение

Мы уважаем вашу конфиденциальность. Приложение «Счётчик дней» разработано таким образом, чтобы ваши пользовательские данные оставались под вашим контролем.

Настоящая Политика конфиденциальности объясняет, какие данные могут обрабатываться при использовании приложения, для каких целей это делается и как эти данные защищаются.

# 2. Какие данные обрабатываются

## 2.1. Данные, которые вы создаёте в приложении

Приложение позволяет вам создавать и хранить собственные записи, включая, например:

- события
- даты
- заметки
- иные данные, которые вы вводите вручную

Эти данные хранятся локально на вашем устройстве.

Если на вашем устройстве включён iCloud, ваши данные могут также синхронизироваться через iCloud в рамках сервисов Apple. Мы не получаем прямой доступ к содержимому ваших записей.

## 2.2. Диагностические и аналитические данные

Для повышения стабильности и удобства использования приложение использует сторонние сервисы Firebase Crashlytics и Firebase Analytics.

В рамках их работы могут автоматически обрабатываться следующие категории данных:

- сведения о сбоях и ошибках приложения;
- техническая информация об устройстве и операционной системе;
- версия приложения;
- данные о взаимодействии с интерфейсом приложения, например:
  - просмотры экранов;
  - нажатия на кнопки;
  - иные события использования, связанные с работой приложения.

Эти данные используются исключительно для:

- диагностики ошибок;
- анализа сбоев;
- улучшения стабильности и работы приложения;
- понимания того, какие элементы интерфейса вызывают ошибки или затруднения в использовании.

## 2.3. Какие данные мы не собираем

Мы не собираем намеренно и не используем для аналитики:

- ваше имя;
- адрес электронной почты;
- номер телефона;
- точное местоположение;
- содержимое ваших записей и заметок для рекламных целей;
- данные для показа таргетированной рекламы;
- данные для отслеживания вашей активности в приложениях и на сайтах третьих лиц.

# 3. Как используются данные

Данные, обрабатываемые приложением, используются только для следующих целей:

- обеспечения основной работы приложения;
- сохранения ваших записей на устройстве;
- синхронизации через iCloud, если эта функция доступна и включена на вашем устройстве;
- диагностики технических проблем;
- улучшения стабильности, производительности и удобства использования приложения.

Мы не продаём ваши данные и не используем их для таргетированной рекламы.

# 4. Передача данных третьим лицам

Приложение может использовать сторонние сервисы, необходимые для его работы и диагностики:

- **Apple iCloud** — для синхронизации ваших данных между вашими устройствами, если вы используете iCloud;
- **Google Firebase Crashlytics** — для получения отчётов о сбоях и ошибках;
- **Google Firebase Analytics** — для получения аналитики использования приложения в обезличенном или агрегированном виде, насколько это предусмотрено настройками сервиса.

Передача данных таким сервисам осуществляется только в объёме, необходимом для выполнения указанных функций.

# 5. Хранение и управление данными

Вы полностью контролируете свои пользовательские записи в приложении и можете:

- создавать, редактировать и удалять события;
- сортировать записи;
- создавать резервные копии;
- восстанавливать данные из резервной копии;
- удалить все данные приложения.

Пользовательские записи хранятся на вашем устройстве и, при использовании iCloud, в вашей учётной записи Apple.

Диагностические и аналитические данные, передаваемые через Firebase, хранятся и обрабатываются в соответствии с политиками и сроками хранения соответствующих сервисов Google Firebase.

# 6. Безопасность данных

Мы принимаем разумные меры для защиты данных, обрабатываемых приложением.

Доступ к пользовательским записям имеют только вы и лица, использующие ваше устройство или вашу учётную запись Apple в рамках функций iCloud.

# 7. Функция обратной связи

Приложение может содержать функцию «Отправить обратную связь». Если вы решите связаться с нами, предоставленная вами информация будет использована только для обработки вашего обращения.

Пожалуйста, не отправляйте конфиденциальную или чувствительную информацию без необходимости.

Новая редакция вступает в силу с момента её публикации. Рекомендуем периодически проверять эту страницу на предмет изменений.
